## Estrutura do Repositório
* `data/`: Bases de dados utilizadas na pesquisa (brutos e processados).
* `scripts/`: Códigos para tratamento de dados e estimação de modelos.
* `docs/`: Documentação e referências bibliográficas.
    * `references/`: Artigos e livros de base.
    * `manuscript/`: Versões do texto da monografia.
* `output/`: Gráficos, tabelas e resultados exportados.
