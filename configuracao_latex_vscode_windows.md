# 📘 Guia Simples: Como usar LaTeX no VS Code (Windows)

Este guia foi feito para escrever documentos em LaTeX.

---

# 🧩 O que você vai instalar

Você precisa de **3 coisas**:

1. Um programa que entende LaTeX (MiKTeX)
2. Um editor de código (VS Code)
3. Uma extensão para facilitar tudo (LaTeX Workshop)

---

# 1 - Instalar o LaTeX (MiKTeX)

## ✔️ Passo a passo

1. Acesse: [https://miktex.org/download](https://miktex.org/download)
2. Baixe o instalador
3. Execute o instalador

Durante a instalação:

* Marque: **Install missing packages on-the-fly → YES**
* Se possível, instale para todos os usuários

---

## 2 - Atualizar o MiKTeX

1. Abra o programa **MiKTeX Console**
2. Vá em **Updates**
3. Clique em **Update now**

---

# 2 - Instalar a extensão LaTeX

1. Abra o VS Code
2. Pressione: **Ctrl + Shift + X**
3. Procure por: `LaTeX Workshop`
4. Clique em **Install**
